# Microcasos Ouro - Estrutural

## Objetivo

Servir como primeira bateria disciplinar do especialista estrutural.

## Lote 01

Este lote cobre os erros mais relevantes já observados no projeto e no fluxo do Orçamentista:

- conflito de FCK
- quadro de armaduras
- revisão conflitante
- laje pré-moldada sem quantitativo fechado
- formas estruturais
- dependência geotécnica

## Arquivos

- `MC-EST-001_conflito_fck.json`
- `MC-EST-002_quadro_armaduras.json`
- `MC-EST-003_revisao_conflitante.json`
- `MC-EST-004_laje_premoldada.json`
- `MC-EST-005_formas_estruturais.json`
- `MC-EST-006_dependencia_geotecnica.json`

## Regra

Cada microcaso deve ser executado como teste unitário do especialista estrutural.

O especialista só passa quando:

- schema permanece válido
- conflito relevante é sinalizado
- hipótese não vira fato
- pergunta HITL aparece quando a base não fecha
