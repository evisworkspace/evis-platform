# Microcasos Ouro

## Objetivo

Servir como dataset disciplinar de validação do EVIS Orçamentista.

## Regra

Cada microcaso deve isolar uma habilidade técnica.

Exemplos:

- quadro de cargas
- detalhe de bloco sobre estaca
- conflito de fck
- ponto de água fria em banheiro
- TUG x TUE
- tabela de esquadrias

## Formato

Usar como base:

- `TEMPLATE_MICROCASO_OURO.json`

## Convenção de IDs

- `MC-EST-###` para estrutural
- `MC-GEO-###` para geotecnico_fundacoes
- `MC-HID-###` para hidraulica_sanitaria
- `MC-ELE-###` para eletrica
- `MC-CIV-###` para civil_execucao
- `MC-CUS-###` para custos_orcamentacao

## Meta inicial

- 50 microcasos por disciplina
- 10 casos médios por disciplina
- 3 projetos completos por disciplina
