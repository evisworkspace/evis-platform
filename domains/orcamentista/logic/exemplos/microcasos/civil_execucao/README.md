# Microcasos Ouro - Civil e Execução

## Objetivo

Servir como primeira bateria disciplinar do especialista civil/execução.

## Lote 01

Este lote cobre os erros mais relevantes para leitura civil e executiva:

- área líquida x área bruta
- desconto de vãos
- separação entre base, regularização e acabamento
- esquadrias sem quadro completo
- preparação/impermeabilização não podem ser omitidas
- risco de dupla contagem de pano/forro

## Arquivos

- `MC-CIV-001_forro_area_liquida.json`
- `MC-CIV-002_revestimento_com_vaos.json`
- `MC-CIV-003_contrapiso_regularizacao_acabamento.json`
- `MC-CIV-004_esquadrias_sem_quadro.json`
- `MC-CIV-005_preparo_impermeabilizacao.json`
- `MC-CIV-006_dupla_contagem_pano.json`

## Regra

Cada microcaso deve ser executado como teste unitário do especialista civil/execução.

O especialista só passa quando:

- schema permanece válido
- medição respeita a lógica executiva
- não mistura camadas de serviço
- pergunta HITL sobe quando o projeto não fecha
