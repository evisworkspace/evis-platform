# Microcasos Ouro - Geotécnico e Fundações

## Objetivo

Servir como primeira bateria disciplinar do especialista de geotécnica e fundações.

## Lote 01

Este lote cobre os erros mais relevantes para fechar fundações com segurança:

- conflito entre tipologia de fundação
- ausência de SPT
- profundidade de estaca não definida
- omissão de serviços auxiliares de solo
- bloco/sapata sem quantitativo definitivo
- risco geotécnico com lençol freático ou solo fraco

## Arquivos

- `MC-GEO-001_tipologia_conflitante.json`
- `MC-GEO-002_ausencia_spt.json`
- `MC-GEO-003_profundidade_indefinida.json`
- `MC-GEO-004_servicos_auxiliares_solo.json`
- `MC-GEO-005_bloco_sapata_quantitativo_aberto.json`
- `MC-GEO-006_risco_geotecnico_agua_solo_fraco.json`

## Regra

Cada microcaso deve ser executado como teste unitário do especialista geotécnico/fundações.

O especialista só passa quando:

- schema permanece válido
- dependência geotécnica é sinalizada quando existir
- hipótese não vira fato
- pergunta HITL aparece quando a base não fecha
