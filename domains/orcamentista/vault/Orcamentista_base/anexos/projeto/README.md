# Anexos de Projeto

Coloque aqui os arquivos principais da obra:

- PDFs de projeto
- pranchas exportadas
- memoriais descritivos ou cadernos de acabamento
- imagens do projeto
- planilhas de quantitativos, listas de servicos ou levantamentos do cliente

Preferencia de leitura do agente:

1. PDF
2. imagens
3. planilhas e tabelas
4. texto

Observacoes:

- o agente deve inspecionar automaticamente esta pasta
- nao dependa de cadastro manual dos arquivos
- `planilha` pode ser XLSX, CSV ou PDF com tabelas
- `memorial` pode ser memorial descritivo, especificacao tecnica, escopo ou caderno de acabamentos
