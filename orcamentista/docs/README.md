# Orçamentista IA - Documentacao Canonica

## Caminho canonico oficial

```text
orcamentista/docs/
```

Esta pasta e a fonte canonica oficial para documentacao especifica do Orçamentista IA.

Use `platform/docs/` para documentacao geral da plataforma EVIS, como navegacao, schema global, direcao visual, mapas amplos da plataforma e decisoes que nao pertencem exclusivamente ao Orçamentista IA.

## Regras de localizacao

- Novos documentos especificos do Orçamentista IA entram em `orcamentista/docs/`.
- Documentos gerais da plataforma EVIS permanecem em `platform/docs/`.
- Nao usar `platform/docs/orcamentista/` como caminho canonico.
- Nao usar `docs/orcamentista/` como caminho canonico.
