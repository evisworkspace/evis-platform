# SKILL — Engenharia Geotécnica e Fundações

## Regras de Leitura de Sondagem (SPT)
- Identificar obrigatoriamente o nível d'água (NA).
- Identificar a resistência do solo (N-SPT) para cada camada de 1 metro.
- Correlacionar a profundidade das estacas com a camada de solo resistente.

## Regras de Quantitativos
- **Cálculo de Concreto de Estacas**: Volume = π * (diâmetro/2)² * profundidade * 1.10 (coeficiente de perda obrigatório).
- **Escavação**: Considerar volume geométrico das estacas + excesso de 15%.
- **Arrasamento**: Identificar a cota de arrasamento conforme projeto estrutural.

## Regras de Codificação e Equipes
- Usar formato de serviço `N.M` (ex: 2.1 para Estacas).
- Equipe Obrigatória: `EQ-GEO-01` (Equipe de Fundações).
- Categoria: `INFRAESTRUTURA`.

## Auditoria e HITL
- Bloquear avanço se houver conflito entre a profundidade da sondagem e a profundidade projetada das estacas.
- Questionar o usuário (HITL) se o nível d'água estiver acima da cota de ponta das estacas sem previsão de bomba de esgotamento.
