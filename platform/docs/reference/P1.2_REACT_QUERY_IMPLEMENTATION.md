2. **Manual Refetch**: User clicks "CARREGAR" button or `refetch()` called
3. **Cache Invalidation**: After sync completes, invalidated tables refetch
4. **Component Mount**: First time hook mounted (if data not in cache)

### When Data is Served from Cache
1. **Data Fresh**: Within 5 minutes of fetch
2. **Tab Switch**: Same data shown from cache
3. **Re-render**: Component re-renders with cached data
4. **Background Refetch**: After 5 mins, silently refetches in background
