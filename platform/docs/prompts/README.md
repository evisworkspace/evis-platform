## Regra de uso

Prompts que forem:

- amplamente reutilizáveis
- independentes de uma sessão específica
- ainda coerentes com o estado atual do projeto

podem ficar aqui.

Prompts que forem:

- de execução única
- já concluídos
- específicos de uma correção pontual
- incompatíveis com o estado atual do projeto

devem ir para `docs/archive/legacy/prompts/`.
