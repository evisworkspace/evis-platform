Guia de Início Rápido - Sistema de Gestão de Obras com IA

1️⃣Como Iniciar o Sistema

Abrir Terminal

Navegue até a pasta do projeto:

cd "C:\Users\User\Evis AI"

Verificar Status (Opcional)

Veja o estado atual antes de iniciar:

.\status.sh

Iniciar Sistema Completo

Execute o script de inicialização:

⚠️ PowerShell vs CMD/Bash

PowerShell (padrão Windows 10/11): use .\start.bat
CMD ou Git Bash: use ./start.bat ou start.bat

.\start.bat

Isso vai iniciar automaticamente:
• Backend (Node.js) na porta 3001
• Frontend (React) na porta 3000

2️⃣Acessar o Sistema

Sistema Pronto!

Após executar o start.bat, abra seu navegador em:

http://localhost:3000

🎨 Frontend

Interface principal

localhost:3000

⚙️ Backend

API e Orquestrador

localhost:3001

3️⃣Verificar Funcionamento

Testar Backend

curl http://localhost:3001/health

Resposta esperada: {"status":"ok","timestamp":"..."}

Testar Frontend

Abra o navegador e verifique se a interface carrega corretamente.

⚠️Problemas Comuns

Erro: "Port 3001 already in use"

A porta já está em uso. Mate o processo:

netstat -ano | findstr :3001
taskkill /PID <PID> /F

Erro: "Cannot find module" ou dependências

Reinstale as dependências:

npm install

Frontend não carrega

Reinicie manualmente:

npm run dev

Backend não responde

Inicie manualmente:

npm run server

Recuperar Sistema

Get-Content RECOVERY.md

📋Fluxo Completo de Uso

  1. Abrir PowerShell ou Terminal na pasta C:\Users\User\Evis AI
  2. Executar .\start.bat (PowerShell) ou start.bat (CMD)
  3. Aguardar mensagens de "Server running on port 3001" e "Local: http://localhost:3000"
  4. Abrir navegador em http://localhost:3000
  5. Usar o sistema normalmente
  6. Para parar: Ctrl+C nos terminais
