```
    │                  sbFetch(servicos),
    │                  sbFetch(pendencias),
    │                  sbFetch(diario),
    │                  sbFetch(notas),
    │                  sbFetch(equipes),
    │                  sbFetch(presenca)
    │             ])
    │             │
    │             └─ setState() with new data ← 6 API CALLS EVERY TIME!
    │
    └─ UI renders
```

### AFTER Implementation
```
main.tsx
    │
    ├─ QueryClient created with cache config
    │  └─ staleTime: 5 min
    │  └─ gcTime: 10 min
    │  └─ retry: 2
    │
    └─ QueryClientProvider wraps App

App Component
    │
    ├─ useSupabaseQuery('servicos', ...) ──→ React Query Cache
    ├─ useSupabaseQuery('pendencias', ...) ──→ React Query Cache
    ├─ useSupabaseQuery('diario', ...) ──→ React Query Cache
    ├─ useSupabaseQuery('notas', ...) ──→ React Query Cache
    ├─ useSupabaseQuery('equipes', ...) ──→ React Query Cache
    └─ useSupabaseQuery('presenca', ...) ──→ React Query Cache
              │
              ├─ Cache HIT (< 5 min) → Return instantly ✅ NO API CALL
              │
              └─ Cache MISS (> 5 min) → Fetch from Supabase
                     │
                     └─ Update cache (5 min timer starts)
                     │
                     └─ syncToSupabase() invalidates ← SMART INVALIDATION
                     │
                     └─ Auto-refetch modified tables
```
