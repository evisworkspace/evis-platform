# PROMPTS LEGADOS

Esta pasta preserva prompts antigos de execução única.

Eles foram mantidos por rastreabilidade histórica, mas não fazem mais parte da área ativa de prompts do projeto.

## Tipos de prompt arquivados aqui

- prompts que já cumpriram sua função
- prompts para correções pontuais
- prompts cuja saída já foi consolidada no repositório
- prompts com premissas que podem não refletir mais o estado atual do sistema

## Regra

Antes de reutilizar qualquer prompt desta pasta, revise se:

- o contexto ainda é válido
- os caminhos citados ainda existem
- as regras descritas ainda batem com o projeto atual
