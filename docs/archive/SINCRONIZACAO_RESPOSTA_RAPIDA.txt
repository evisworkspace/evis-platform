## 🎯 Resposta Direta

**Você precisa de 3 correções sincronizadas:**

### 1️⃣ **Melhorar Prompt IA** (Diario.tsx)
Forçar a IA SEMPRE retornar `data_inicio` e `data_fim`
- **Por quê?** Cronograma precisa dessas datas para desenhar as barras

### 2️⃣ **Validar Datas em confirmIA()** (Diario.tsx)
Se IA não retornar datas, preencher automaticamente
- **Por quê?** Garantir que 100% dos serviços têm datas válidas

### 3️⃣ **Invalidar React Query Cache** (Diario.tsx + App.tsx)
Quando aplicar IA, dizer ao React Query "refetch servicos"
- **Por quê?** Cronograma pega dados do cache, precisa atualizar
